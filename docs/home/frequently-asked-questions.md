# 常见问题汇总

## 1. 为什么我所有的任务点都已完成，但还是没有满分
> 在观看视频时，由于平台的问题，可能在观看结束后仍然显示观看时长为0，从而导致所有任务点已完成，但分数却没有100%。不用担心，你可以重新观看视频来获取分数。我们的脚本也将在未来更新选择视频一键挂课的功能
